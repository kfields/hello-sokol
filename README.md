[Sokol](https://github.com/floooh/sokol)

## Getting Started

```bash
git clone --recursive https://github.com/kfields/hello-sokol
cd hello-sokol

mkdir build
cd build
mkdir debug
cd debug
cmake -G "Visual Studio 17 2022" -DCMAKE_BUILD_TYPE=Debug ../..
```
